? HOW IT WORKS

Process

  1. Upload: CSV or Excel file
  2. Select: Columns to group by
  3. Configure: Prefix (hf_, fac_, etc.)
  4. Generate: Unique hf_uid codes
  5. Download: Updated file

Grouping

  • Select multiple: adm1, adm2, adm3, hf
  • Unique combo: Each combination = 1 ID
  • Example: Western|Urban|Freetown|HospA → hf_1
  • Same combo: Gets same ID across rows

Output

  • New "hf_uid" column: hf_1, hf_2...
  • Unique ID per combination
  • Mapping table included
  • Download as Excel or CSV
